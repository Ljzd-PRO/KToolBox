# Pawchive について

[Pawchive](https://pawchive.pw/) は KToolBox v1 が対応する唯一のバックエンドです。KToolBox は公開 API からクリエイターと投稿を検出し、Pawchive 専用ファイルホストからファイルをダウンロードします。

## 既定のエンドポイント

| 用途 | 既定値 |
| --- | --- |
| 公開 API | `https://pawchive.pw/api/v1` |
| クリエイターのアイコンとバナー | `https://pawchive.pw` |
| 投稿ファイル | `https://file.pawchive.pw/data/...` |

API リクエストがアカウントセッションを送信することはありません。`downloader.session_key` を設定した場合は、ファイルダウンロードのリクエストだけに `session` Cookie として送信されます。

## 対応する API の範囲

KToolBox は公開された OpenAPI 文書のうち、`cookieAuth` を必要としない 14 の操作をすべて実装しています。クリエイターと投稿の検出、プロフィール、告知、ファンカード、リンク、投稿詳細、ファイルハッシュ検索、投稿のフラグ状態と送信、改訂、コメント、アプリケーションバージョンが含まれます。

Pawchive アカウントへのログインが必要な 5 つのお気に入り操作は意図的に除外しています。

## 責任ある利用

適用される法律、プラットフォームの規約、クリエイターの権利、ストレージ上限を守ってください。クリエイターを初めて試すときは、`--length` で同期件数を制限し、ファイルサイズの上限も設定してください。
