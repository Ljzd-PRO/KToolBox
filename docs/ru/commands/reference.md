# Справочник команд

Выполните `ktoolbox COMMAND --help`, чтобы получить точную справку Cyclopts. Имена команд и параметров используют дефисы; прежнее написание с подчёркиваниями по-прежнему разбирается скрытыми командами совместимости.

## Глобальные параметры

| Параметр | Значение |
| --- | --- |
| `-h`, `--help` | Вывести справку напрямую, без программы просмотра. |
| `--version` | Вывести установленную версию KToolBox. |
| `--install-completion` | Установить автодополнение для обнаруженной оболочки. |
| `--config PATH` | Выбрать файл проекта. Порядок разрешения: этот параметр, `KTOOLBOX_PROJECT_CONFIG`, затем `./ktoolbox.toml`. |
| `--verbose` | Включить диагностические журналы. |
| `--quiet` | Скрыть ход выполнения и обычные журналы. |
| `--plain` | Принудительно использовать стабильный построчный прогресс. Для вывода не в TTY и `NO_COLOR` он включается автоматически. |
| `--no-color` | Отключить цвета ANSI. |

Запуск `ktoolbox` без аргументов выводит корневую справку и успешно завершается.

## Дерево команд

| Команда | Назначение |
| --- | --- |
| `download` | Скачать одну публикацию или выбранную редакцию. |
| `sync [TARGET ...]` | Синхронизировать явно заданных авторов либо всех включённых авторов списка, если цель не указана. |
| `creator list/add/remove/enable/disable/search` | Управлять списком или искать авторов Pawchive. |
| `post show/search` | Просмотреть публикацию или искать публикации автора. |
| `config edit/example/validate/path` | Изменить или просмотреть окружение и конфигурацию проекта. |
| `site-version` | Вывести версию приложения Pawchive. |
| `webui [PROJECT_DIR]` | Запустить необязательную панель HeroUI для одного проекта. |

## `download`

Укажите URL публикации Pawchive либо все параметры `--service`, `--creator-id` и `--post-id`.

| Аргумент или параметр | Тип | По умолчанию | Значение |
| --- | --- | --- | --- |
| `POST` | строка | не задано | URL публикации или редакции Pawchive. |
| `--service` | строка | не задано | Служба автора. |
| `--creator-id` | строка | не задано | ID автора. |
| `--post-id` | строка | не задано | ID публикации. |
| `--revision-id` | строка | не задано | Выбрать эту редакцию из списка. |
| `-o`, `--output`, `--path` | путь | `.` | Корневой каталог вывода. |
| `--dump-post-data` / `--no-dump-post-data` | логическое | включено | Сохранить проверенные метаданные в `post.json`. |

`download` намеренно не применяет правила исключения списка.

## `sync`

Каждый `TARGET` может быть URL автора Pawchive, идентификатором `service:id` или псевдонимом из списка. Явные цели выполняются, даже если отключены в списке. Без целей выполняются все включённые авторы.

| Аргумент или параметр | Тип | По умолчанию | Значение |
| --- | --- | --- | --- |
| `TARGET ...` | строки | включённый список | Ноль или более авторов. |
| `--service` + `--creator-id` | строки | не задано | Добавить явного автора; оба значения обязательны. |
| `-o`, `--output`, `--path` | путь | `.` | Корневой каталог вывода. |
| `--save-creator-indices` | логическое | выключено | Атомарно сохранить индекс автора после успешного производства. |
| `--mix-posts` / `--no-mix-posts` | логическое | конфигурация окружения | Переопределить `job.mix_posts`. |
| `--start-time`, `--start` | дата | не задано | Включительная нижняя граница даты публикации, `YYYY-MM-DD`. |
| `--end-time`, `--end` | дата | не задано | Включительная верхняя граница даты публикации, `YYYY-MM-DD`. |
| `--offset` | целое | `0` | Индекс первой публикации. |
| `--length` | целое | все | Максимум принятых публикаций для каждого автора. |
| `--keywords` | повторяемая строка | конфигурация окружения | Включить заголовки, содержащие любое значение. |
| `--keywords-exclude` | повторяемая строка | конфигурация окружения | Устаревший совместимый ввод для исключения заголовков. |

`job.creator_concurrency` ограничивает производство авторов, а `job.count` — общие обработчики файлов.

## `creator`

| Команда | Аргументы и параметры | Значение |
| --- | --- | --- |
| `creator list` | `--json` | Вывести записи списка как таблицу Rich или JSON. |
| `creator add TARGET` | `--alias NAME`, `--disabled` | Добавить URL или `service:id`. |
| `creator remove TARGET` | | Удалить по псевдониму, URL или идентификатору. |
| `creator enable TARGET` | | Включить сохранённого автора. |
| `creator disable TARGET` | | Отключить сохранённого автора. |
| `creator search` | `--name`, `--creator-id`/`--id`, `--service`, `--dump`, `--json` | Отфильтровать публичный список авторов. |

## `post`

| Команда | Аргументы и параметры | Значение |
| --- | --- | --- |
| `post search` | `--creator-id`/`--id`, `--name`, `--service`, `-q`/`--query`, `-o`/`--offset`, `--dump`, `--json` | Искать публикации выбранных авторов. Прямой запрос API содержит не менее 3 символов, смещение API делится на 50. |
| `post show SERVICE CREATOR_ID POST_ID [REVISION_ID]` | `--dump`, `--json` | Показать метаданные текущей публикации или выбранную редакцию. |

Без `--json` команды запросов намеренно не выводят текст публикации в таблицах терминала.

## `config`

| Команда | Значение |
| --- | --- |
| `config path` | Вывести разрешённый путь проекта без переноса строк. |
| `config validate` | Проверить версию схемы, уникальность авторов, типы и области правил, условия и регулярные выражения. |
| `config example` | Сформировать все параметры dotenv из строк документации модели конфигурации. |
| `config edit` | Открыть необязательный редактор Urwid и проверить данные перед сохранением. |

## `webui`

Установите `ktoolbox[webui]` перед использованием этих команд. Стандартная команда принимает каталог проекта и после предупреждения создаёт отсутствующий `ktoolbox.toml`. Без настроенного аккаунта терминал выводит имя `admin` и случайный пароль для этого процесса; параметры WebUI позволяют задать постоянные данные.

| Аргумент или параметр | По умолчанию | Значение |
| --- | --- | --- |
| `PROJECT_DIR` | `.` | Фиксированный проект синхронизации, обслуживаемый процессом. |
| `--host` | `webui.host` | Переопределить интерфейс прослушивания. |
| `--port` | `webui.port` | Переопределить порт TCP. |
| `--no-open` | выключено | Не открывать локальный URL в стандартном браузере. |
| `webui hash-password` | | Дважды запросить скрытый ввод в терминале и вывести хеш Argon2id. |

Встроенный сервер выводит предупреждение безопасности HTTP. Перед доступом за пределами localhost ознакомьтесь с [руководством по WebUI](../webui.md).

## Псевдонимы совместимости

Эти псевдонимы скрыты из справки и выводят одно предупреждение об устаревании за вызов:

| Прежнее имя | Замена |
| --- | --- |
| `download-post` | `download` |
| `sync-creator` | `sync` |
| `search-creator` | `creator search` |
| `search-creator-post` | `post search` |
| `get-post` | `post show` |
| `config-editor` | `config edit` |
| `example-env` | `config example` |

## Коды завершения и потоки

| Код | Значение |
| --- | --- |
| `0` | Успех. |
| `1` | Ошибка удалённой службы, автора или загрузки, включая частичный успех нескольких авторов. |
| `2` | Недопустимые аргументы или конфигурация. |
| `130` | Прерывание пользователем. |

Таблицы и JSON используют stdout. Журналы, ход выполнения, предупреждения и ошибки используют stderr.
