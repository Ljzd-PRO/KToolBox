# Справочник конфигурации

Имена переменных окружения начинаются с `KTOOLBOX_` и соединяют вложенные поля модели через `__`. Типы `path`, `set` и `list` разбираются Pydantic; для коллекций в dotenv используйте массивы JSON.

## Корневые параметры

| Поле | Тип | По умолчанию | Описание |
| --- | --- | --- | --- |
| `ssl_verify` | логическое | `True` | Проверять сертификаты TLS для API и загрузок. |
| `json_dump_indent` | целое | `4` | Отступ вывода JSON. |
| `use_uvloop` | логическое | `True` | Использовать установленный uvloop в Unix или winloop в Windows. |

## `api`

| Поле | Тип | По умолчанию | Описание |
| --- | --- | --- | --- |
| `scheme` | `http` / `https` | `https` | Схема URL API. |
| `netloc` | строка | `pawchive.pw` | Сервер API. |
| `statics_netloc` | строка | `pawchive.pw` | Сервер статических ресурсов авторов. |
| `path` | строка | `/api/v1` | Корневой путь API. |
| `timeout` | дробное | `5.0` | Тайм-аут каждого запроса в секундах. |
| `retry_times` | целое | `3` | Дополнительные попытки при ошибках транспорта, `429` и `5xx`. |
| `retry_interval` | дробное | `2.0` | Задержка между попытками API в секундах. |

Группа API намеренно не содержит ключ сеанса.

## Проектный `ktoolbox.toml`

Проектный документ отделён от конфигурации окружения. Его путь разрешается из глобального `--config`, затем `KTOOLBOX_PROJECT_CONFIG`, затем `./ktoolbox.toml`.

### Корень

| Поле | Тип | По умолчанию | Описание |
| --- | --- | --- | --- |
| `schema_version` | литерал `1` | `1` | Версия схемы проекта; другие значения отклоняются. |
| `creators` | массив таблиц | пусто | Сохранённый список авторов. |
| `blockers` | массив таблиц | пусто | Упорядоченные спецификации правил. |

### Запись автора

| Поле | Тип | По умолчанию | Описание |
| --- | --- | --- | --- |
| `service` | непустая строка | обязательно | Служба Pawchive. |
| `creator_id` | непустая строка | обязательно | ID автора Pawchive. |
| `alias` | строка / не задано | не задано | Уникальная цель CLI; `:` запрещён. |
| `enabled` | логическое | `True` | Включение в `sync` без целей. |

### Запись правила

| Поле | Тип | По умолчанию | Описание |
| --- | --- | --- | --- |
| `id` | идентификатор | обязательно | Уникальный ID из букв, цифр, `.`, `_` или `-`. |
| `type` | строка | `field-match` | Зарегистрированная реализация. Неизвестные типы отклоняются. |
| `enabled` | логическое | `True` | Участвует ли правило. |
| `scope.mode` | `global` / `creators` | `global` | Применить глобально или к выбранным идентификаторам. |
| `scope.creators` | список `service:id` | пусто | Обязателен и не пуст для области авторов; запрещён для глобальной. |
| `options.rule` | группа условий | обязательно для `field-match` | Корневое рекурсивное правило. |

Группы условий используют `kind = "group"`, `mode = "any"` или `"all"`, непустой список `conditions` и необязательный `negate`. Условия полей используют `kind = "field"`, безопасный точечный `field`, один из `contains`, `equals`, `regex` или `exists`, а также необязательные `case_sensitive`, `negate` или `expected`. Операторам кроме `exists` нужен непустой список `values`; `exists` запрещает `values`.

## `webui`

Эти параметры нужны только при установке `ktoolbox[webui]` и необязательны для запуска: пустое имя становится `admin`, а если обе формы пароля пусты, сервер создаёт случайный пароль, выводит итоговые данные в терминале и хранит их только в этом процессе.

| Поле | Тип | По умолчанию | Описание |
| --- | --- | --- | --- |
| `host` | строка | `0.0.0.0` | Интерфейс прослушивания HTTP. Вне доверенной LAN предпочитайте `127.0.0.1`. |
| `port` | целое | `8789` | Порт HTTP от 1 до 65535. |
| `open_browser` | логическое | `True` | Открыть локальный URL панели после запуска. |
| `username` | строка | пусто | Необязательное имя; при пустом значении используется `admin`. |
| `password_hash` | секретная строка | пусто | Предпочтительный постоянный хеш Argon2id. |
| `password` | секретная строка | пусто | Открытый резерв; создаётся при запуске, если обе формы пусты. |
| `max_active_tasks` | целое | `2` | Параллельные задачи верхнего уровня, от 1 до 16. |
| `session_idle_hours` | целое | `24` | Срок сеанса от последнего использования. |
| `session_absolute_hours` | целое | `168` | Максимальный срок сеанса от входа. |

Все имена используют обычный префикс, например `KTOOLBOX_WEBUI__PASSWORD_HASH`. Создайте хеш командой `ktoolbox webui hash-password`; заключите его в кавычки в dotenv, поскольку Argon2 содержит символы `$`. Параметры `--host`, `--port` и `--no-open` переопределяют настройки для одного запуска.
