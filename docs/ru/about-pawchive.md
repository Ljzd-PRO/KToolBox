# О Pawchive

[Pawchive](https://pawchive.pw/) — единственный сервер, поддерживаемый KToolBox v1. KToolBox использует его публичный API для поиска авторов и публикаций, а затем скачивает файлы с выделенного файлового сервера Pawchive.

## Стандартные адреса

| Назначение | Значение по умолчанию |
| --- | --- |
| Публичный API | `https://pawchive.pw/api/v1` |
| Значки и баннеры авторов | `https://pawchive.pw` |
| Файлы публикаций | `https://file.pawchive.pw/data/...` |

Запросы API никогда не содержат сеанс учётной записи. Если настроен `downloader.session_key`, он отправляется как cookie `session` только в запросах на скачивание файлов.

## Поддерживаемые операции API

KToolBox реализует все 14 операций опубликованного документа OpenAPI, не требующих `cookieAuth`: поиск авторов и публикаций, профили, объявления, карточки подписчиков, ссылки, сведения о публикациях, поиск по хешу файла, проверку и отправку отметок публикации, редакции, комментарии и версию приложения.

Пять операций с избранным требуют входа в учётную запись Pawchive и намеренно исключены.

## Ответственное использование

Соблюдайте применимые законы, условия платформы, права авторов и ограничения хранилища. При первой проверке автора используйте ограниченную синхронизацию (`--length`) и ограничения размера файлов.
