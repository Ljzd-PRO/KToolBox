# KToolBox

KToolBox — асинхронный загрузчик командной строки, проектная панель HeroUI и типизированный клиент Python для общедоступных данных [Pawchive](https://pawchive.pw/). Версия 1 поддерживает только Pawchive и требует Python 3.10–3.14.

## Возможности

- Скачивание одной публикации или параллельная синхронизация списка авторов.
- Применение упорядоченных глобальных или привязанных к автору правил исключения до создания заданий на скачивание.
- Продолжение частичных загрузок и пропуск существующих файлов.
- Фильтрация по дате, заголовку, шаблону имени файла и размеру.
- Раздельное управление выводом обложек, вложений, изображений содержимого, метаданных и внешних ссылок.
- Постоянная WebUI на семи языках для настройки проекта, редактирования списка авторов и правил исключения, запросов Pawchive и управления жизненным циклом задач.
- Все 14 публичных операций OpenAPI Pawchive через проверенные модели Pydantic.

Операции с избранным, требующие аутентификации учётной записи, намеренно не реализованы. Настроенный ключ сеанса загрузчика отправляется только на файловый сервер.

## Установка

`pipx` позволяет изолировать приложение:

```bash
pipx install ktoolbox
```

Установите необязательный терминальный редактор и поддержку оптимизированного цикла событий:

```bash
# Linux / macOS
pipx install "ktoolbox[urwid,uvloop]" --force

# Windows
pipx install "ktoolbox[urwid,winloop]" --force
```

При необходимости отдельно установите браузерную панель:

```bash
pipx install "ktoolbox[webui]" --force
```

## Быстрый старт

```bash
# Просмотр команд и параметров.
ktoolbox -h
ktoolbox download -h

# Скачивание одной публикации.
ktoolbox download https://pawchive.pw/fanbox/user/6570768/post/1836570

# Начните с одной публикации, прежде чем синхронизировать больший диапазон.
ktoolbox sync https://pawchive.pw/fanbox/user/6570768 --length 1
```

![Обзор команд KToolBox](../assets/cli-overview.png)

Сохраните несколько авторов и синхронизируйте все включённые записи:

```bash
ktoolbox creator add fanbox:123 --alias studio-a
ktoolbox creator add patreon:456 --alias studio-b
ktoolbox sync
```

При повторных запусках существующие файлы пропускаются. Незавершённый файл с настроенным временным суффиксом возобновляется, если файловый сервер поддерживает диапазоны байтов.
