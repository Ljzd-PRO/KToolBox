    Coomer¶

    KToolBox 支持从 Coomer.st / Coomer.su / Coomer.party 下载

    你需要通过 dotenv文件 .env 或 prod.env 或系统环境变量来修改配置： 

    # Coomer API
KTOOLBOX_API__NETLOC=coomer.st

# 用于从 Coomer 服务器下载文件
KTOOLBOX_API__FILES_NETLOC=coomer.st
